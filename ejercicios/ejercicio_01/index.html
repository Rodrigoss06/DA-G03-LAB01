Proyectos Favoritos

Clon de Spotify

Un clon de la interfaz de Spotify desarrollado con Next.js, Tailwind CSS y Astro. Este proyecto recrea la experiencia de usuario de Spotify, incluyendo funcionalidades como navegación entre secciones, reproducción de música y manejo de listas de reproducción. Perfecto para mostrar habilidades en desarrollo front-end y uso de tecnologías modernas.

  • Next.js
  • Tailwind CSS
  • Astro

Search Index de Películas y Videojuegos

Un proyecto de indexación y búsqueda de películas y videojuegos desarrollado con Next.js, Tailwind CSS y PostgreSQL. Este proyecto permite a los usuarios buscar y filtrar títulos de una base de datos extensa, mostrando habilidades en desarrollo full-stack, diseño de interfaces modernas y manejo de bases de datos relacionales.

  • Next.js
  • Tailwind CSS
  • Postgresql
